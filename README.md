# Fourdigital Website

Маркетинговый сайт на Next.js (App Router) с мультиязычным контентом (`ru`, `uz`, `en`) и SEO-страницами.

## Требования

- Node.js 20+
- npm 10+

## Локальный запуск

```bash
npm install
npm run dev
```

Приложение будет доступно на `http://localhost:3000`.

## Скрипты

- `npm run dev` — запуск dev-сервера
- `npm run build` — production-сборка
- `npm run start` — запуск production-сборки
- `npm run lint` — проверка линтером
- `npm run test` — запуск тестов

## Структура

- `src/app` — маршруты и страницы
- `src/components` — UI и секции
- `src/lib` — контент, SEO-данные и утилиты
- `public` — статические ассеты
